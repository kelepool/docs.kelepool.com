# 设置算力报警

### **邮箱登录-设置算力报警**

* 选择需要设置收款地址的子账号
* 点击左侧菜单“账户设置”，确定您要配置的币种，点击“算力报警”
* 设置监控间隔时间、算力跌幅、离线数量。输入报警联系人信息，填写邮箱或手机。当算力跌幅或离线数量达到设置的阈值，系统将发送报警提醒。
